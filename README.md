# Postman API Testing Portfolio

This repository contains example API testing scenarios created using Postman.

Purpose:
Demonstrate practical REST API testing skills including request validation, response verification and status code checks.

Tested Endpoints:
- GET users
- GET user by ID
- POST create user
- PUT update user
- DELETE user

Validation Performed:
- Status code verification
- Response body validation
- JSON structure validation
- Negative test scenarios

Tools Used:
- Postman
- REST APIs
- JSON validation

Example Test Types:
- Functional API testing
- Positive test cases
- Negative test cases
